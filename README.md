## 新闻发布管理系统
项目描述：采用React+Redux+Webpack技术栈实现的SPA应用。实现了登录、权限管理、角色管理、新闻撰写-审核-发布-管理流程、游客访问等功能。
- 通过利用React-Router的多级路由，实现重定向未登录用户到登录界面，在用户界面配合后端实现权限管理
- 抽出公共组件在功能相似的界面复用、用Redux在邻居组件间传递状态
- 利用Axios拦截器+Redux实现多界面统一Loading状态管理
